## Eco-Friendly Product Website 🌱

A responsive website promoting plantable seed-based products with Bootstrap carousel and nature-inspired design.

### Tech Stack
- HTML, CSS, JavaScript
- Bootstrap 5

### Features
- Smooth scroll navigation
- Carousel slider for eco-products
- Responsive design
